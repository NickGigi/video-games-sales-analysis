Exploratory data analysis of global video game sales trends, regional preferences, and publisher performance using Python and Pandas.
# Video Games Sales Analysis

## Overview
Exploratory data analysis of a 16,000+ entry video games sales dataset, uncovering global market trends, cultural differences, and genre evolution over time.

## Key Findings
- PS2 was the strongest performing platform globally
- North American markets favor Action/Shooter games; Japan favors RPGs
- Sports games peaked in 2009 then declined ~70% by 2015
- Top publishers (Nintendo, EA, Activision) rank consistently across NA and EU, but NA sales volumes are significantly larger

## Technologies Used
- Python, Pandas, Matplotlib, Seaborn

## Dataset
- 16,621 rows, 11 columns (cleaned to 16,268 rows)
- Source: Video Games Sales dataset (CSV)
